# Auth Microservices

> The project is simple fastAPI auth microservices backend web app.

#### Dependency

- python3
- Postgresql
- Docker

###### Instructions:

```bash
git clone https://github.com/mbrsagor/auth.git
cd auth
virtualenv venv --python=python3.10
source venv/bin/activate
pip install -r requirements.txt
```

### Features:
- User signUp with
- User SignIn with JWT
- User profile
- Update user profile
